# Privacy Policy

Effective date: July 15, 2026

Extract Behance Moodboard is a local Codex skill that helps users archive images from projects they have saved to a Behance moodboard.

## Data the skill processes

The skill may process:

- a Behance moodboard URL supplied by the user;
- project titles, gallery URLs, and image URLs visible in the user's normal Behance session;
- a local JSON manifest created for the extraction;
- image files downloaded to a folder selected by the user; and
- local extraction summaries and error messages.

## How data is handled

The skill runs in the user's Codex environment. The project maintainers do not operate a backend service for this skill, do not receive the user's manifests or downloaded images, and do not add analytics or telemetry.

The skill sends normal web requests only to services required for the requested workflow, including Behance, Behance image delivery domains, and the local browser integration selected by the user. OpenAI, Adobe, Behance, GitHub, and other third-party services process data under their own terms and privacy policies.

## Credentials

The skill must not inspect, copy, store, or transmit browser cookies, passwords, authentication tokens, or other credentials. It uses only the access already available through the user's normal browser session.

## Storage and deletion

Manifests, downloaded images, and summaries remain in the user's selected local folders. The user controls retention and deletion. The maintainers have no remote copy to delete.

## User responsibilities

Users must only process content they can lawfully access and are permitted to save. Copyright and other rights remain with the original creators and rights holders.

# Support

Use GitHub Issues for bug reports, compatibility problems, and feature requests:

https://github.com/ilariaislwj-lang/extract-behance-moodboard/issues

Include:

- your operating system and Python version;
- whether you used a moodboard URL or the current browser tab;
- the number of projects visible in the saved-project grid;
- the error summary from `_extract-summary.json`; and
- a minimal example that does not expose private project names or URLs.

Do not post passwords, cookies, session tokens, private manifests, private moodboard URLs, or downloaded copyrighted images in an issue.

This open-source project does not provide guaranteed response times. Security or privacy concerns should be reported with the minimum information needed to reproduce the problem.
